# MindMate App

### MindMate is a supportive mental health companion app designed to help users track their moods, chat with an AI assistant, and encourage self-care practices. Built with Ionic + Angular and powered by Firebase and Claude AI, MindMate provides an easy-to-use and accessible platform for emotional wellbeing.

### Features
- User authentication (sign up, login, guest mode)
- Chatbot powered by Claude 3
- Mood tracker calendar
- Progressive Web App (PWA) support
- Secure user data storage with Firebase Firestore
- Real-time chat history saving
- Responsive design for mobile and desktop

---

### Technologies Used
- Ionic Framework (Angular)
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting
- Firebase Cloud Functions (Claude Proxy API)
- Claude 3 API (Anthropic)
- Node.js 18 for backend functions
